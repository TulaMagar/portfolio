What is comment for?

Programming languages are formal languages that are used to express computations. People reading your code should be able to understand what the program is doing in natural (human) language. In a computer program, a comment is a text that is only intended for human readers and is completely ignored by the interpreter.



Why is the comment used?

  • Comments help the reader (coder or non-coder) to understand the code.
  • Comments help to explain the function, Class, and others.
  • Write a document of the program using the comment.

    • Types of comments in python:

    There are two types of comments in python: single-line and multiple lines comments. Python understands that # denotes comment and ignores the rest of the statement from the # when the program runs.
